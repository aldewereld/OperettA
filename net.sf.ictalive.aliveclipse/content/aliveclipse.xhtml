ALIVEclipse

The ALIVEclipse tool allows one to design and run ALIVE systems. The tool guides designers through the process of designing the various models that are required to run an ALIVE system.

On first starting the ALIVEclipse tools, you are greeted by the welcome screen. This screen contains information about ALIVEclipse, the methodology of ALIVE, and tutorials for the tools included in ALIVEclipse. Click the "Go to workbench" button (the button with the arrow on it) to close the welcome screen (or click the small 'X' on the welcome-tab at the top of the screen). You can always re-open the welcome screen (to browse the tutorials) by selecting it in the "Help" menu.

To start making an ALIVE system, one should first create an ALIVE project. Creating an ALIVE project is done by either right clicking in the 'ALIVE Navigator' view (on the left-hand of the screen) or by selecting the 'New' option in the 'File' menu. Both options are shown in the figure below.

 
Figure 1. Creating a new ALIVE project.

If you selected the 'New' option in the 'File' menu, you will be presented with a list of various types of projects and files that can be created. The option to create an ALIVE project is found under the listing 'ALIVE', as shown in figure 5. After clicking 'Next' you will be presented with the ALIVE project wizard.
If you choose the other option of creating an ALIVE project, the ALIVE project wizard will be shown right away.

Give your project a meaningful (easy to remember) name in the field 'Project Name'. We will use 'ALIVE_Tutorial' for the remainder of this document. WARNING: Do not use spaces in project names.

For most users the default location will suffice, so click 'Finish' to complete the process. If required, you can change the storage location of your ALIVE projects by de-selecting the 'Use default location' option, and inputting your own preference.

Your new project is located on the left-hand plane, which is called the 'ALIVE Navigator', see the figure below.

This view will show all of your ALIVE projects, allowing you to quickly switch between different projects you are authoring. Clicking the small arrow next to the project will open the project to show its contents (or collapse it if the project was already opened).
